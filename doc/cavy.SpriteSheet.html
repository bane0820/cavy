SpriteSheet

  • Extends
  • cavy.Sprite

new SpriteSheet(obj, pattern, param)

canvasにスプライト画像を配置するオブジェクトクラス
Parameters:
Name Type Description
obj Object widthとheightを持つオブジェクト
pattern Object スプライトアニメーションパターン
param Object 初期パラメータ
Example
   var s = new SpriteSheet(img, {
		walk: [0,0,3,2],	//(0,0)から3フレームを１回繰り返しアニメーションさせます
		sample: [オフセットx,オフセットy,フレーム数,繰り返し回数]
	});
